## Figure Standards

All figures follow `PLOT_CONFIG` from `scripts/generate_baseline_v23_figures.py`:
- 300 DPI, serif font, 12pt minimum
- Wong 2011 colorblind-friendly palette
- Both PNG (raster) and PDF (vector) formats
- Labeled axes with units
